# Student Management System

## Project Overview
Student Management System is a backend-based application built using Python and MySQL to manage student records efficiently. The project performs CRUD operations and demonstrates database connectivity, modular backend structure, and basic data management functionality.

## Features
- Add student records
- View all students
- Search student by roll number
- Update student details
- Delete student records
- MySQL database integration

## Technologies Used
- Python
- MySQL
- SQL
- GitHub

## Project Structure
- `app.py` - main application logic
- `database.py` - database connection setup
- `schema.sql` - database schema
- `requirements.txt` - project dependencies
- `README.md` - project documentation

## How to Run
1. Install Python on your system
2. Install MySQL
3. Run the `schema.sql` file in MySQL
4. Install required package using:
   `pip install -r requirements.txt`
5. Update MySQL username and password in `database.py`
6. Run the project using:
   `python app.py`
